# nodejs-crawling

---

## Project Introduce 
TypeScript와 HTML, jquery를 이용해서 크롤링을 구현했다. 
Typeorm, cheerio등을 사용했다.

- 인하광장 최다추천 랭킹 크롤링
- CRUD
- EC2 내부 디비 사용
